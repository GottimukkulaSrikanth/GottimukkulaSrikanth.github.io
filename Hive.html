Apache Hive is a data warehousing infrastructure built on top of Hadoop.
It provides a HiveQL to query and manage large datasets stored in Hadoop Distributed File System (HDFS).
Hive translates queries into MapReduce jobs, making it easier for users to interact with big data using familiar SQL syntax.

Difference between Hive and traditional relational databases
Traditional relational databases that use a schema-on-write approach where as Hive uses a schema-on-read approach.
In Hive, the structure of the data is defined at the time of querying rather than during data ingestion.
