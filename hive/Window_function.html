Window Function

RANK() and DENSE_RANK(): Assign a value to each row within a partition based on a specified column's values.

RANK() leaves gaps in case of ties, while
DENSE_RANK() Assigns the same rank to equal values.
ROW_NUMBER(): Assign a unique number to each row within a partition.

Query :

    
RANK() OVER (PARTITION BY Department ORDER BY SalesAmount DESC) AS Rank
FROM Sales;
